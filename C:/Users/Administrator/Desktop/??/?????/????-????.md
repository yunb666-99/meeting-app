# 服务器部署说明

## 你需要准备什么

| 东西 | 说明 |
|------|------|
| **一台 Linux 云服务器** | 推荐阿里云/腾讯云，4核8G内存，50M带宽，Ubuntu 24.04 |
| **一个域名** | 比如 `meeting.你的公司.com`，需要备案 |
| **会操作服务器的人** | 帮你执行下面几步命令 |

---

## 快速部署（4步）

### 第 1 步：买服务器 + 域名

打开阿里云或腾讯云，买一台服务器，配置选：
- CPU：4核 或更多
- 内存：8GB 或更多
- 系统：Ubuntu 24.04
- 带宽：50Mbps
- 在安全组里开放端口：80、443(TCP+UDP)、7881(TCP)、50000-50100(UDP)

域名买好后，在云平台完成备案（阿里云/腾讯云有免费备案协助）。

### 第 2 步：上传文件到服务器

把整个 `服务器文件` 目录上传到服务器的 `/opt/meetingapp/`

### 第 3 步：运行部署脚本

SSH 登录服务器：
```bash
ssh root@你的服务器IP
cd /opt/meetingapp
chmod +x scripts/setup.sh
./scripts/setup.sh
```

脚本会自动问你域名和服务器IP，然后一键装好所有东西。

### 第 4 步：验证

打开浏览器访问：
- `https://你的域名/admin` → 管理后台登录页
- `https://你的域名/api/docs` → API 文档

## 服务器日常操作

```bash
# 查看服务是否在运行
docker compose ps

# 查看日志（出问题用）
docker compose logs -f

# 重启
docker compose restart

# 备份数据库
./scripts/backup.sh
```

## 忘记密码怎么办

```bash
docker compose exec backend node -e "
const bcrypt = require('bcryptjs');
const { PrismaClient } = require('@prisma/client');
const prisma = new PrismaClient();
async function reset() {
  const hash = await bcrypt.hash('你的新密码', 10);
  await prisma.user.update({ where: { account: '2942146423' }, data: { passwordHash: hash } });
  console.log('密码已重置');
}
reset();
"
```
